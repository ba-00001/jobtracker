# AI Usage Documentation - PostgreSQL Practice Version

## Tools Used

- ChatGPT

## What AI Was Used For

AI assistance was only used for:

- README wording and organization
- Optional mock data ideas

## Key Prompts

- "Help me organize a README for a Flask and MySQL job application tracker project."
- "Generate a few mock job application tracker sample records I can use for testing."

## AI-Generated Output Used

- Draft wording for setup instructions in the README
- Example mock data ideas for users, companies, jobs, and applications

## Changes Made After AI Output

- Rewrote the final README so it matched the PostgreSQL practice structure
- Adjusted any sample data ideas to fit the schema used in the project
- Removed anything that was not needed for the assignment

## What I Learned

- AI is useful for speeding up documentation and brainstorming sample data
- It is still important to verify that generated content matches the real code and schema
- Keeping AI use limited made it easier to document exactly what it helped with
