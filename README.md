# Job Tracker Workspace

This repository now contains two separate versions of the project:

- `mysql-app/`: the official course-submission version that follows the assignment requirement to use MySQL
- `postgres-app/`: a parallel PostgreSQL version for practice and comparison

## Which One To Submit

Submit the code from `mysql-app/` for your class project unless your professor explicitly approves PostgreSQL.

## Folder Guide

- `mysql-app/README.md`: setup for the MySQL course version
- `postgres-app/README.md`: setup for the PostgreSQL practice version

## Important Note

The course specification requires MySQL, so `mysql-app/` should remain your primary graded project.
